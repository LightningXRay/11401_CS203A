| 特性 | Stack |
|------|-------|
| 結構特性 | 後進先出（LIFO） |
| 插入 / 刪除 | 只在頂端（Top） |
| 常見操作 | push / pop |
| 存取方式 | 不支援隨機存取 |
| 存取位置 | 只能存取頂端 |
| 常見用途 | 函式呼叫、括號檢查、DFS |
