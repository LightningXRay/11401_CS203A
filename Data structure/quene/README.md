| 特性 | Queue |
|------|-------|
| 結構特性 | 先進先出（FIFO） |
| 插入位置 | 尾端（Rear） |
| 刪除位置 | 前端（Front） |
| 存取方式 | 不支援隨機存取 |
| 常見操作 | enqueue / dequeue |
| 常見用途 | 排程、緩衝區、BFS |
