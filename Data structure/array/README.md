特性:
同質性（Homogeneous）:陣列中所有元素資料型態相同

例如（C）：int a[5] = {1, 2, 3, 4, 5};

連續記憶體配置

陣列元素在記憶體中是連續儲存

好處：存取快

壞處：大小不易改變

固定大小（Static Size）

多數語言中，陣列建立後大小不能改

例：int a[10]; → 永遠只能放 10 個元素
